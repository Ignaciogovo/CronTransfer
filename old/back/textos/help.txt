Comandos:
cron_insert 
        + s  -->  realizar un insert de un servicio de backup completo con guía. Es el parametro por defecto
        + c  --> Realizar una insercción de datos de una conexión con guía
        + cf --> Realizar una insercción rápida de datos de conexion, es necesario otros parametros:
            IP, puerto(d='Por defecto'/ cualquier puerto) usuario p/k --> password/key
            Ejemplo: cron_insert cf 192.168.1.4 d  nombre_usuario p 

backup_daily
        Parametros: id_conexion hora /ruta/origen /ruta/final log(Y/N) sobrescribir(Y/N)
        Ejemplo: backup_daily 7 3 /backups_bbdd/ultimobackup.sql /backups Y N
            --Se enviaría el archivo /backups_bbdd/ultimobackup.sql a la ruta /backups de la conexión con id 7 
            todos los días a las 3 AM con archivo log y sin sobrescribir la backup en el servidor remoto


cron_select 
        + c --> muestra por pantalla las conexiones
        + s --> muestra por pantalla los servicios backups
        + a --> muestra por pantalla las dos consultas anteriores --> Es el parametro por defecto


cron_run
        + id --->( id representa el codigo identificador del servicio)
        Ejemplo: cron_run 7 --> Ejecuta el servicio con id 7



cron_delete
        + s --> Borra un servicio con guía
        + c --> Borra una conexión con guía
        + sf  --> Borra servicio a partir del parametro dado (Es necesario el id del servicio que se desea borrar)
            Ejemplo: cron_delete sf 7 --> Borra el servicio con id 7
                Borrar varios servicios: Ejemplo: cron_delete sf 4 5 6 7 --> Borra los servicios 4 5 6 7
        + cf --> Borra conexión a partir del parametro dado
            Ejemplo: cron_delete sf 4 --> Borra la conexion con id 4
                Borrar varias conexiones: Ejemplo: cron_delete sf 4 5 6 7 --> Borra las conexiones 4 5 6 7


cron_help --> Muestra el archivo ayuda por pantalla
